# React to Weather App
### Getting Started
1. Clone locally using

    ```git clone https://github.com/carlosmed/react-to-weather```

2. Install NPM packages

    ```npm install```

3. Run Development or Production

    ```npm run dev```

    or

    ```npm run prod```
